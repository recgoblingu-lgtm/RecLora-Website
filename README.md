# RecLora Website

Official website for RecLora - A community-driven revival of Rec Room.

## 🚀 Deploy to Cloudflare Pages

1. Go to https://dash.cloudflare.com/
2. Click "Pages" → "Create a project"
3. Select "Connect to Git"
4. Authorize GitHub and select this repository
5. Leave build settings empty (no build needed)
6. Deploy!

## 📁 Files

- `index.html` - Homepage
- `download.html` - Downloads
- `news.html` - News
- `roadmap.html` - Roadmap
- `faq.html` - FAQ
- `team.html` - Team
- `contact.html` - Contact
- `css/style.css` - Styling

## 🎨 Customization

- Edit HTML files directly to change content
- Update colors in `css/style.css`
- Add logo to `assets/` folder
